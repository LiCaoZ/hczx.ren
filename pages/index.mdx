# HCZX.REN

嗨，这里是 HCZX.REN。

HCZX.REN 的意思可以理解为“合川中学的人”、“与合川中学有关的人”

HCZX.REN 为 [重庆市合川中学](https://zh.wikipedia.org/wiki/%E5%90%88%E5%B7%9D%E4%B8%AD%E5%AD%A6) 的学生、教职员工与校友免费提供子域名。简言之，只要你可以证明你与合川中学存在上述关系，均可免费获得一个子域名。

你可以选择为你解析 NS 记录，这样你就可以完全控制此子域名的解析；你亦可以只解析 CNAME 或 A 记录，不过这样在部分场景可能受限。

你可以在 [这里](/how-to) 了解如何获取，在 [这里](/tos) 了解子域名使用规范。
